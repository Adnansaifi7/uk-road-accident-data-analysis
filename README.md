# UK Road Accident Data Analysis

## Project Overview
This project analyzes the UK Road Accident dataset using Big Data and Data Mining techniques. The objective is to identify accident patterns, uncover hidden insights, and understand the factors contributing to road accidents. The project includes data preprocessing, exploratory data analysis (EDA), visualization, and predictive modeling.

## Objectives
- Clean and preprocess accident data.
- Perform exploratory data analysis (EDA).
- Identify accident trends over time.
- Analyze accident severity and contributing factors.
- Build predictive models using data mining techniques.
- Visualize findings with charts and graphs.

## Dataset
The project uses the UK Road Accident dataset, which contains information such as:
- Accident Date and Time
- Location
- Road Type
- Weather Conditions
- Light Conditions
- Speed Limit
- Vehicle Type
- Casualty Information
- Accident Severity
